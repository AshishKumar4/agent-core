# W3 integration requests

W3 owns the Facet, Operation, and authenticated Slot runtime. It deliberately does not
edit aggregate barrels or the definition, authority, and invocation domains. These
fragments identify the exact integration contracts those workstreams must satisfy.

The W3 gateway never accepts caller-assembled Grants, epochs, leases, placement, trust,
or invocation evidence. W0 must not export the host constructors until W2 and W6 wire
the production ports and their adversarial suites pass.
